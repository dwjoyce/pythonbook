A project to create a textbook for teaching Python 3 on Rasberry Pi 2B computers

Useful commands
---------------

Install sphinx:

```bash
pip3 install sphinx
```

Build a pdf (when inside root of this repository):

```bash
make latexpdf
```

To test python code:

```bash
make pythontest
```

The pdf will then be at `build/latex/pythonbook.pdf`.
